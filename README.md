#  FAEX Hub — React Router

Trabalho avaliativo desenvolvido com React + TypeScript utilizando React Router e Styled Components.

## Objetivo

Implementar rotas e páginas de Login e Cadastro seguindo layout do Figma, com organização adequada de componentes e estilos.

---

##  Tecnologias Utilizadas

- React
- TypeScript
- React Router DOM
- Styled Components
- Vite

---

##  Estrutura do Projeto
src/
├── components/
│ ├── Logins/
│ │ ├── Login.tsx
│ │ └── Login.Styles.ts
│ └── Cadastro/
│ ├── Cadastro.tsx
│ └── Cadastro.Styles.ts
├── App.tsx
└── main.tsx

##  Rotas Implementadas

| Rota | Descrição |
|------|-----------|
| `/` | Dashboard (em desenvolvimento) |
| `/login` | Página de Login |
| `/cadastro` | Página de Cadastro |

---

##  Navegação

As páginas de Login e Cadastro possuem navegação entre si utilizando Link do React Router.

---

## Estilização

Toda a estilização foi feita utilizando Styled Components, sem uso de CSS externo ou Tailwind.
